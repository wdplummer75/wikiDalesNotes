Macintosh Setup for Brother HL-5470DW monochrome laser printer

This topic outlines the steps to install the "biogradprinter1" printer on Macintosh computers.

For command line steps to install department printers on Linux computers see Installing Linux Printers using the Command Line.

There is also a color printer available for graduate student use called "biostatcolor5". See Install biostatcolor5 Printer on Macintosh Computers.

The printers "biostatcolor5" and "biogradprinter1" are exclusively for the use of graduate students.
Open System Preferences and click on Printers & Scanners

Click the + icon to add a new printer

Since this is a networked printer, click the IP tab. Enter the configuration information as shown and click Add. The printer's hostname is "biogradprinter1.dhcp.mc.vanderbilt.edu".
